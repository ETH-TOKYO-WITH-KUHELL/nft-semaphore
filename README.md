## installation guide
```shell
npm install --save-dev hardhat
npx hardhat
npm install @openzeppelin/contracts
```

## how to use hardhat
- https://dashboard.alchemy.com/
```shell
export ALCHEMY_API_KEY=
export MUMBAI_PRIVATE_KEY=
export METADATA_URL=
npx hardhat run scripts/deploy.ts --network mumbai
```
